Ethonwork Bailing (Lark)

A powerful, fast and flexible GPT-Like large AI language model for everything you want based on LLaMA. The goal is to make a GPT for everyone. Accurate, efficient and fast.

Introduction

Bailing (Lark) is a large GPT-Like language model, which based on LLaMA / Alpaca. It is trained based on 7,500,000,000 parameter dataset. Thanks to the open-source project llama.cpp, which is working on 4-bit quantized FP-16/FP-32 LLaMA/Alpaca based language models to work smoothly on low-end PCs. Bailing is trained to work in English, Chinese, Korean and Japanese in different branches and we may merge them into one for a better experience in the future.

Performance

The development of Bailing is aware of system performance. In the case, you only need a 2-core CPU and 4GB RAM space to run the model, no GPUs are required. This mean you can run the entire Bailing model and chat smoothly locally on you own PC even without Internet connections. Behind the amazing performance, we tried the 3.5 billion and 7.5 billion of parameter dataset, then quantized them to 4-bit FP-16 to reduce system requirements as much as possible. Also, thanks to the open-source communities, we can remove redundant thesaurus and data in the model. All of these have achieved the ultimate performance of Bailing.

Development

The development of Bailing is based on the open-source model Meta LLaMA and Stanford Alpaca. We re-trained a version of the model with the same dataset from GPT-4 and another with a daily chat dataset. Also, we made a relive of our recent experiments on native Chinese/Japanese/Korean support named CANGJIE. 

Branching

Bailing is divided into 3 branches, which focus on different directions.


A : The goal of this branch is to provide a stable, powerful and always-useable language model based on LLaMA/Vicuna original model branches. Provides amazing English support. With the original data from GPT-4, A Branch models are more accurate and powerful. We can reach 95% of the GPT performance but the hardware requirements are way much less than GPT.


B : The B Branch is more aggressive with daily-chat-training based Koala model. Trying to provide a more flexible and creative experience. With 7.9 billion lines of data, B Branch models can reach 85% of the FPT-3.5 performance. As our experimental model branch, lots of new features and datasets will be added into the models as quickly as possible.


C : The C Branch, codename CANGJIE, is a relive of our recent experiments on Chinese language. Models in C Branch can understand and answer Chinese sentences natively and fluently. CJK characters and UTF-8 is fully supported. With 700+ thousand lines of data in Chinese, CANGJIE has became the world's first big language model in Chinese.
